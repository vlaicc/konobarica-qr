QR Code Generator

Unesi URL tvoje GitHub Pages stranice:


Napomena: Prvo postavi stranicu na GitHub Pages, pa onda unesi URL i napravi QR kod.

Kako postaviti na GitHub Pages:

  1. Idi na github.com i napravi nalog (besplatno)
  2. Napravi novi repository sa imenom "konobarica-qr"
  3. Upload-uj index.html fajl
  4. Idi u Settings > Pages
  5. Izaberi "main" branch i "/ (root)" folder
  6. Klikni Save - dobices URL
  7. Vrati se ovde i napravi QR kod sa tim URL-om
